**This sample app demonstrates how to integrate Daimo Pay.**

For details, [see our docs](https://paydocs.daimo.com).

We're using the Daimo Pay SDK: [@daimo/pay](https://www.npmjs.com/package/@daimo/pay)


## Quick start

```
npm i
npm run dev
```
